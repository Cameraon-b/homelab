# Work Completed

## 1. Windows Server Media Preparation
  Downloaded Windows Server evaluation ISO from [Microsoft Evaluation Center](https://www.microsoft.com/evalcenter?utm_source=chatgpt.com).

### Issues Encountered
  * Windows hid file extensions, resulting in:
      windows-server-2022.iso.iso

### Resolution

  * Enabled file extension visibility in Windows
  * Corrected filename

____________________________________________

## 2. Secure File Transfer to Nora
  Used OpenSSH `scp` to transfer installation media from Windows workstation to Nora.

### Issues Encountered  
  * Folder paths with spaces caused PowerShell parsing errors
  * Incorrect Linux home directory typo caused transfer failures

### Resolution
  * Used quoted paths in PowerShell
  * Corrected destination path
  * Successfully transferred ISO to:
      /var/lib/libvirt/boot/
____________________________________________

## 3. Virtual Network Deployment
  Attempted to use default libvirt network.

### Issues Encountered
  * No default network present
  * Missing `libvirt-dnsmasq` user/group
  * Network XML templates absent

### Resolution
  * Created custom NAT network XML manually
  * Reinstalled virtualization networking components
  * Created missing service accounts
  * Successfully defined and activated persistent network

### Result
  default network = active
  autostart = enabled
  persistent = yes

Guest subnet established:
  192.168.122.0/24

## 7. Host Dual-Boot Recovery
  While Windows 11 updates were installing on secondary workstation:

### Issues Encountered
  * Unexpected reboot into Kali Linux
  * Windows Boot Manager temporarily disappeared
  * GRUB defaulted to Kali

### Resolution

  Booted into Kali Linux.
  Ran:
    sudo update-grub
  
  Windows Boot Manager rediscovered.
  Set Windows as default boot entry.
  Confirmed subsequent Windows updates completed successfully.

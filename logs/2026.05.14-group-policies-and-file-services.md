# Lessons Learned

* Active Directory containers differ from Organizational Units.
* Group Policy targeting depends on object placement.
* UNC paths are required for centralized resource delivery.
* User and computer policies target different AD objects.
* Domain password and lockout policies must be configured at domain root.
* Windows may appear as a single OS but internally behaves as many cooperating identities and services.
* Event Viewer provides deep visibility into authentication activity.
